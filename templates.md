# Templates — Every Shape You'll Need

*The Solo Path · companion to Stages 1–5*

> One rule governs this whole file: **the shapes are borrowed; the
> content never is.** A Foundry whose content is borrowed is someone
> else's Foundry with your name on it. Copy the structures freely.
> Fill them only with what you've actually lived.

Copy these into whatever tool you'll actually open every week.

------------------------------------------------------------------------

# Stage 1 Shapes

## Story Card

```
STORY: (short title, in your words)
Date captured:
Context: (which part of life — job, community, family, side project)

What happened:
(the concrete event — names, stakes, specifics)

What it revealed:
(the transferable lesson)

Where else I've seen it:
(even a hunch)
```

## Open Questions Page

```
OPEN QUESTIONS
(the home for uncertainty — never resolve a question just because
uncertainty feels uncomfortable)

Questions I can't answer yet:
-

Hunches awaiting evidence:
- I have a hunch that ___ — but I've only seen it ___ times,
  in ___ context(s).

Things I'm tempted to declare true — parked instead:
-
```

## Monthly Pattern Check (30 minutes)

```
PATTERN CHECK — (month)

1. Reread every story card.
2. What keeps repeating? (new hunches → Open Questions page)
3. How is each existing hunch doing?
   - confirmed by new stories from DIFFERENT contexts?
   - contradicted by anything?
4. Does my North Star question still fit my best stories?
   (if the best ones escape it, widen it)
5. Stories captured this month: ___ / contexts represented: ___
```

# Stage 2 Shapes

## Pattern Entry

```
PATTERN: (your name for it — words you'd say to a friend)
Status: named (promoted from hunch on ___)

The claim, in one sentence:

Evidence (stories that support it — list 3+, different contexts):
1. (story title — context)
2. (story title — context)
3. (story title — context)

Counter-evidence (be honest; "none yet" is a valid entry):

What new stories should watch for:
```

# Stage 3 Shapes

## Experiment Record

```
EXPERIMENT: (what you're testing, in one line)
Pattern being tested:
Date:

What I tried:
What I expected:
What actually happened:
What surprised me:

What this does to the pattern:
(strengthens / sharpens / contradicts — and how)
```

## Principle Entry (your working canon)

```
PRINCIPLE: (one sentence, descriptive — what the evidence shows,
not who you want to be)

Earned via pattern:
Evidence:
- (story — context)
- (story — context)
- (story — context)
- (experiment)

First written: (date)
Last challenged: (date — a principle nobody has challenged in a year
deserves a deliberate look)
```

## Question Library Entry

```
QUESTION: (the question, exactly as you ask it)

When I reach for it:
What it tends to unlock:
Where it came from: (story or moment that taught it to you)
```

# Stage 5 Shapes

## Decision Log Entry

```
DECISION: (what you decided, one line)
Date:

Why:
Evidence that supported it:
What would make me revisit it:
```

## Quarterly Review (60–90 minutes)

```
QUARTERLY REVIEW — (quarter)

Reread: stories, patterns, principles, moves, open questions.

- Which open questions resolved themselves through practice?
- Which hunches earned promotion? (different contexts? 3+ stories?)
- Which patterns or principles deserve DEMOTION or revision?
  (if the answer is never "any," the system is agreeing, not filtering)
- Which questions should be retired?
- What's missing from the story library? (whole areas of life
  going uncaptured?)
- One experiment to run next quarter:
```

## Your Own Promotion Criteria (write this when the one rule needs elaborating)

```
MY PROMOTION PATH

Observation → Story → Pattern → Experiment → Practice/Move → Principle
→ Constitution (rare)

At each stage, the evidence I require before promoting:
- hunch → pattern: ___
- pattern → principle: ___
- principle → constitution: ___

My default answer at every stage: NOT YET. KEEP OBSERVING.
```

# The Rhythm (all stages)

| Cadence | Ritual | Time |
|---|---|---|
| Weekly | One story card | 10 min |
| Monthly | Pattern check | 30–45 min |
| Quarterly (Stage 5+) | Full review | 60–90 min |
| Rarely | Constitution | only when evidence demands |
