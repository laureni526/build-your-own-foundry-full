# The Council

<!--

**Sequence:** [your voices in order] → Weaver

**Job:** Take the idea once. Apply each voice skill in turn, holding each
output. When every voice has spoken, apply the weaver skill to their
outputs and return its synthesis as the primary answer — with the
underlying voices available below it if I want to see the raw
disagreement.

**Never does:** Skip a voice to save time, collapse two voices into one
pass, or let the Weaver run before every voice has actually spoken. If I
only asked about one angle, don't convene the whole council — hand it to
the single voice instead.
